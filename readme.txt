超台本　「ヒ・ト・ミ」 ver1.0

OS：Windows2000/XP/Vista/7　日本語版のみ対応
CPU：Pentium3 800MHz以上推奨
メモリ：256MB以上推奨
HDD：400MB以上
DirectX：9.0c以上
サウンド：ogg形式
画面解像度：1024×768

製作者：岡崎潤
製作ツール：吉里吉里2/KAG3

■はじめに
　このたびは、本作品を手に取っていただき、誠にありがとうございます。
本作品は劇団InnocentSphereが2009年に公演された「ヒ・ト・ミ」の台本と公演の写真を
もとに構成されました。
エンディングまでのお付き合いいただき、もしInnocentSphereの舞台にご興味がわきましたら、
劇場まで足をお運びいただき、生の舞台を体感してみてください。

■インストール
zip形式で圧縮されています。解凍後、「hitomi.exe」をダブルクリックして起動させてください。

■アンインストール
フォルダごと「ごみ箱」に移動させて消去します。セーブデータは「savedata」フォルダに保存されています。
